# 🌍 Global Universities Explorer (UniExplorer)
### *Your Free Digital Consultant for Global Education*

UniExplorer is a premium, data-driven platform designed to democratize access to international education. By providing transparent data on universities, scholarships, and visa pathways, we empower students to transition from "dreaming" to "doing" with absolute clarity.

---

## 🚀 Core Features

### 1. **Smart University Explorer**
- **Dynamic Search**: Filter thousands of programs by country, degree level (Undergraduate, Masters, PhD), field of study, and language.
- **Tuition & Budget Tracking**: Compare annual costs in USD to find programs that fit your financial profile.
- **Detailed Profiles**: Access direct links to university websites and view ranking scores.

### 2. **The Student Portal (Gated Access)**
- **Eligibility Checker**: A personalized tool where students enter their GPA and preferences to see which top-tier programs they qualify for.
- **Personalized Recommendations**: An intelligent matching engine that ranks universities based on the user's academic profile, budget, and ranking preferences.
- **Scholarship Finder**: A curated database of global financial aid opportunities (e.g., Rhodes, Gates Cambridge, DAAD) with deadline alerts.

### 3. **Interactive Success Roadmap**
- **Country-Specific Pathways**: Tailored guidance for the UK, USA, Canada, Australia, Germany, France, and Turkey.
- **Resource Hub**: One-click access to official testing (IELTS/TOEFL) and government visa portals.
- **Step-by-Step Milestones**: Detailed phases covering Research, Documentation, Application, and Visa Interview.

## 📝 Installation & Setup

1. **Clone & Environment**:
   ```bash
   pip install -r requirements.txt
   ```
2. **Database**:
   - Create a DB named `universities_db` in PostgreSQL.
   - Run `psql -d universities_db -f schema.sql`.
3. **Run**:
   ```bash
   python app.py
   ```
